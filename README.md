# FrameDrop-RL

A reinforcement learning model that drops unnecessary frames depending on the network state in the end device.

## Usage

### train
```bash
python train.py
```
### run
```bash
python run.py
```

## Dependency

read requirement.txt
```bash
pip install -r requirements.txt
```
